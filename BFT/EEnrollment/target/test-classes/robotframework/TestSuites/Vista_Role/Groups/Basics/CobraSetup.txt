2.Terminate EE with Termination Reasons_InvoluntaryMisConduct
    [Documentation]    Author : *Srikanth G*
    ...
    ...    Test Case Title : *Terminate EE with Termination Reasons_Involuntary_MisConduct*
    ...
    ...    Test Case Description : Login as vista and adds Employee and switch to HR role then terminate employee with "Involuntary due to gross misconduct" as reason.
    ...
    ...    Test Steps :
    ...
    ...    1) Launch eeshrel and login as vista admin (bfautomation/Password1)
    ...
    ...    2) Perform Group Search
    ...
    ...    3) Naviagte to COBRA Setup > Termination Reason > Yes > Retirement Reasons > No > Death Reasons > Yes > SAVE
    ...
    ...    4) Navigate to Users->Members
    ...
    ...    5) Clcik on Add Member and Enter all required information and click save
    ...
    ...    6) Perform Member Search
    ...
    ...    7) Select Member Options and Select 'Open Member In HR Role'
    ...
    ...    8) Perform Employee Termination with " Involuntary due to gross misconduct" as reason
    ...
    ...    9) Verify employee is terminated and is not COBRA eligible. The cancellation/termination \ rules are applied.
    ...
    ...    10) Logout Vista Admin
    [Tags]    QCOE
    Login as vista admin with valid credentials
    Perform Group Search    td:automationgroup
    Cobra setup configuration    Termination Reasons    Yes    yes
    Cobra setup configuration    Retirement Termination Reasons    No    yes
    Cobra setup configuration    Death Termination Reason    Yes    yes
    Navigate To    Users    Add Member
    Add Member In Vista    td:addemployeewithmedicalforcobra    lastname
    Open Member Option From Vista Role    Open in HR Role
    Perform Employee Termination    d:currentdate    Involuntary due to Gross Misconduct
    Close HR Admin
    Logout Vista Admin
